NextGen Tracker

A mobile academic planning app for tracking terms, courses, and assessments.

About the app

NextGen Tracker helps students organize their academic schedule in one place. Users can create and manage terms, add courses, track assessments, and generate an upcoming assessments report with a run timestamp. The app also includes account login with hashed password protection and optional reminders for key dates.

Key features

  • Create, edit, and delete terms, courses, and assessments
  • Search/filter and generate an upcoming assessments report (multi-row, multi-column, timestamped)
  • Secure login with password hashing
  • Optional notifications for course/assessment dates
